Name

*CACHE-TABLE-QUERIES-DEFAULT* — Specifies the default behaviour for caching of attribute types.

Variable

Value Type

A valid argument to the action parameter of cache-table-queries, i.e. one of T, NIL, :flush.

Initial Value

nil

Description

Specifies the default behaivour for caching of attribute types. Meaningful values are T, NIL and :flush as described for the action argument to cache-table-queries.

Examples

None.

Affected By

None.

See Also

cache-table-queries

Notes

None.
