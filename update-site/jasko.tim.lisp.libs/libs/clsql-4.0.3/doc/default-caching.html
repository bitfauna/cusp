*DEFAULT-CACHING*
Prev Object Oriented Data Manipulation Language (OODML) Next

Name

*DEFAULT-CACHING* — Controls the default caching behavior.

Variable

Value Type

Boolean

Initial Value

T

Description

This variable stores the default value of the CACHING keyword for the select.

Examples

	(let ((*default-caching* nil)))
	  (select 'foo))

	;; is equivalent to

	(select 'foo :caching nil)

Affected By

None.

See Also

select

Notes

This is a CLSQL extension to the CommonSQL API. CommonSQL has caching on at all times.
